# Basic Artisan Commands

Other basic useful commands

```php
php artisan clear-compiled // clear compile file
php artisan down // put application in maintenance mode
php artisan up // put it back live
php artisan env // current php envoronment
php artisan help
php artisan optimize // clear and refresh configuration & route files

php artisan cache:clear
php artisan view:clear
php artisan session:table // automatic session table

```
